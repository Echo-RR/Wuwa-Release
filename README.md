## Getting Started

1. Download `WUWA Launcher.exe`.
2. Place it in a folder where your Windows account can write files.
3. Run the launcher and follow the on-screen instructions.
4. If Windows displays a security prompt, confirm that the file came from this
   official repository before continuing.

## Access

An active WUWA license is required. Access is linked to one PC.

If the launcher says that your PC is not licensed, copy the Hardware ID shown
by the launcher and send it to the administrator through the official support
channel.

## System Requirements

- Windows 10 or Windows 11, 64-bit
- Internet connection
- Supported Wuthering Waves installation
- Permission to run the launcher and write files in its folder

## Updates

The launcher checks for a newer public version when it starts. Keep the
launcher in a writable folder so it can apply an available update.

Only download updates from this repository or the official support channel.

## Troubleshooting

When requesting support, include:

- Windows version
- The exact message shown by the launcher
- What happened immediately before the issue
- A screenshot of the message, if possible

Do not post private license information or other sensitive data publicly.

## Notice

This repository is the public distribution page for the launcher. Its internal
source code and service infrastructure are maintained privately.

Use the launcher only where you have permission to use it. Availability and
compatibility may change after game or service updates.
